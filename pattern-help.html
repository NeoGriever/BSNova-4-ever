Hilfe zu Patterns
Die Patterns in BSNova dienen dazu, Serien und Filme entsprechend zu benennen und in Ordner zu sortieren. Hierbei können mehrere "Platzhalter" genutzt werden, um daraus gültige Pfadangaben zu erzeugen.

Das Einstellungs-Fenster von BSNova4ever besitzt eine Live-Vorschau für die Formulierung von Patterns. Somit kann man immer sofort sehen, wie sich ein Pattern auf eine Episode oder einen Film auswirkt.

Es gibt folgende Pattern-Platzhalter:
Pattern Beschreibung
%serie% Der Name der entsprechenden Serie
%seasonNN% Die Staffel-Nummer. Diese wird auf 2 Stellen aufgewertet. Das heißt, Staffel 9 wird zu 09.
%seasonN% Die Staffel-Nummer ohne Aufwertung auf 2 Stellen.
%episodeNN% Die Episoden-Nummer. Diese wird auf 2 Stellen aufgewertet. Das heißt, Episode 4 wird zu Episode 04.
%episodeN% Die Episoden-Nummer ohne Aufwertung auf 2 Stellen.
%name% Der Name der Episode. Hierbei wird vorzugsweise der deutsche Name gewählt. Ist kein deutscher Name vorhanden, wird der englische Name gewählt.
%nameE% Der englische Name der Episode. Sollte kein englischer Name vorhanden sein, wird der Platz nicht mit dem deutschen Namen ersetzt.
%nameG% Der deutsche Name der Episode. Sollte kein deutscher Name vorhanden sein, wird der Platz nicht mit dem englischen Namen ersetzt.
Es ist möglich, innerhalb des Patterns Backslashes zu nutzen, welche zur Erzeugung von Ordner dienen. Ein Beispiel:

%serie%\%seasonNN%\%episodeNN% - %name%.mp4

Dadurch wird im Ordner "SERIENNAME" der Ordner "STAFFELNUMMER" erzeugt, worin dann die Episoden eingeordnet werden.

Beispiel-Ergebnis: (Downloadpfad)\Beispielserie\02\06 - Beispielepisode.mp4

Es ist wichtig, dass am Ende IMMER die Dateiendung angehängt wird! Es wird empfohlen, diese mit .mp4 zu benennen.
